NIO Framework

O objetivo da NIO é criar o melhor framework atual, focado na interface do usuário, padronizar projetos e com fidelidade com o design, por que dos melhores frameworks tem um design de 1990? Por que não criar um framework leve, rápido e bonito?

Nosso objetivo é criar uma comunidade open-source, queremos criar um dos melhores frameworks css, pretendemos criar uma biblioteca Javascript e outras ferramentas para ajudar desenvolvedores #GoNIO
